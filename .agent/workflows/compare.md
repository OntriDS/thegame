# COMPARE COMMAND

**Purpose:** Compare multiple items/options/approaches and provide a structured matrix or table comparison.

**What COMPARE does:**
1. **Identifies Comparison Items:** Takes 2 or more items to compare (can be files, approaches, systems, features, etc.)
2. **Creates Comparison Matrix:** Generates a structured table showing differences and similarities
3. **Analyzes Key Factors:** Compares relevant aspects like performance, maintainability, complexity, features
4. **Provides Recommendations:** Gives clear guidance on which option is best for specific use cases

**Usage:** `COMPARE [item1] vs [item2] vs [item3]` - Compare multiple items in a structured table format.

**Example:** `COMPARE localStorage vs KV vs database` - Compare storage approaches with pros/cons matrix.
