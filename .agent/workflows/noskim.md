# NOSKIM COMMAND

**Purpose:** Force complete document reading without any skimming, scanning, or pattern-matching behavior.

**What NOSKIM does:**
1. **Forces Complete Reading:** Must READ every single line of the specified document(s)
2. **Prevents Surface-Level Understanding:** Cannot rely on document structure to guess content
3. **Enforces Deep Comprehension:** Must understand context, relationships, and nuances
4. **Eliminates Assumption-Based Responses:** Cannot make claims without having read completely
5. **Accountability for Full Understanding:** Must be able to answer questions about any part

**Usage:** `NOSKIM [document-path]` - Forces complete reading of the specified document.
