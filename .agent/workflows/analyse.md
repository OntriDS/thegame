# ANALYSE COMMAND

**When user types ANALYSE, you must:**
1. **Investigate system architecture and data flow**
2. **Find issues and inconsistencies** 
3. **Research business logic and workflows**
4. **Report on performance, scalability, user experience and security**

**Analysis Areas:**
- System Architecture, Data Integrity, Business Logic, Performance, User Experience, Security, Technical Debt

**Output Format:** Executive Summary, Detailed Analysis, Action Items
