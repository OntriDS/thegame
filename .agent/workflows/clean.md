# CLEAN COMMAND

**Purpose:** Automatically clean old code, remove unused functionality, and ensure DRY principles.

**What CLEAN does:**
1. **Remove Debugging Code:** Remove temporary console.log() statements
2. **Remove Unused Functionality:** Identify and remove dead code
3. **Enforce DRY Principles:** Consolidate duplicate code patterns
4. **System Rules Compliance:** Ensure UI components use Z_INDEX_LAYERS constant
5. **Code Quality:** Fix linting errors, ensure proper TypeScript types
